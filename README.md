Emergency Medical Portal is a web-based system designed to streamline emergency medical record access securely and efficiently. This project was developed during the ACM Hackathon at PDEU by our team: **Hit Sheth, Kavan Trivedi, Raj Gandhi, and Patel Priyank**.

## ğŸ” What does it do?

In critical situations, quick access to patient history can save lives. Our platform allows authorized personnel to securely retrieve medical records using Aadhaar numbers. This helps doctors make informed decisions instantly, ensuring better emergency care.

- âœ… **Secure Admin Login**: Only authorized personnel can access records.
